KeymouseGo - Windows 7 Compatible Release
==========================================

This is a Windows 7 compatible build of KeymouseGo.

Requirements:
- Windows 7 SP1 or later (also works on Windows 8/10/11)
- No additional dependencies required

Usage:
1. Double-click KeymouseGo.exe to start the application
2. Use F6 to start script execution
3. Use F9 to stop script execution  
4. Use F10 to toggle recording

Command Line Usage:
  KeymouseGo.exe script.json -rt 3
  
  Options:
    scripts     Path to script file(s)
    -rt, --runtimes   Number of times to run the script (default: 1)

Build Information:
- Python 3.8.10
- PySide2 5.15.2.1
- PyInstaller 6.17.0

For more information, see README.md or visit:
https://github.com/taojy123/KeymouseGo
